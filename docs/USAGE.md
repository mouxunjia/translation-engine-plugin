# User Manual

## Quick Start
1. Copy plugin to .obsidian/plugins/
2. Enable in Obsidian settings
3. Press Ctrl+Shift+T to translate

## Shortcuts
- Ctrl+Shift+T : Translate
- Ctrl+Shift+C : Cultural check
- Ctrl+Shift+R : Daily review
- Ctrl+Shift+O : Tools panel

## Languages
zh-CN, zh-TW, en-US, ja-JP, ko-KR, fr-FR, de-DE, es-ES, ru-RU, ar-SA
