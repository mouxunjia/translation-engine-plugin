# Translation Engine Plugin

Professional translation plugin for diplomatic scenarios.

## Features
- Multi-engine translation (Ollama + Cloud)
- Cultural sensitivity detection
- Daily review with push notifications
- 7 integrated tools

## Quick Start
1. Copy to .obsidian/plugins/
2. Run: pip install -r requirements.txt
3. Enable plugin in Obsidian

## Shortcuts
- Ctrl+Shift+T : Translate
- Ctrl+Shift+C : Cultural check
- Ctrl+Shift+R : Daily review
